# Assets — Pokémon Landing

## Logo oficial

Drop aqui:
- `pokemon-logo.png` — logo oficial Pokémon (amarelo + borda azul)
- Recomendado: PNG transparente, ~1200px largura
- Alternativas: `.svg` (melhor) ou `.webp`

Se o arquivo nao existir, a landing usa um fallback CSS (Bungee Shade) automaticamente.

## Backgrounds opcionais

Se quiser dropar imagens de fundo personalizadas:
- `bg-mewtwo.jpg` — para secao Mewtwo (atmosfera tempestuosa)
- `bg-mew.jpg` — para secao Mew (rosa/psiquico suave)
- `bg-herois.jpg` — para secao Herois & Viloes (palco/dramatico)

Tamanho recomendado: 1920x1080, JPEG 80% quality (~150KB cada).
